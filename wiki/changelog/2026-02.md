### Current config state

The configuration is organized into focused modules:

| Module | Purpose |
|---|---|
| `config/rendering.lua` | GPU frontend, FPS, power preference |
| `config/font.lua` | Maple Mono + Nerd Font fallback, size, line height |
| `config/appearance.lua` | Catppuccin Mocha/Latte, dark/light auto-switching |
| `config/window.lua` | Padding, decorations, tab bar, blur, opacity |
| `config/keys.lua` | CMD key bindings mapped to Neovim/tmux commands |
| `config/background.lua` | Wallpaper + gradient overlay system, NOAA live wallpaper |
| `config/events.lua` | User-var handlers for COLOR_SCHEME and ZEN_MODE |

Utilities:

| Module | Purpose |
|---|---|
| `utils/helpers.lua` | `is_dark()`, `get_random_entry()` |
| `utils/keys.lua` | `cmd_key()`, `cmd_to_tmux_prefix()`, `multiple_actions()` |
