# Share Skill

A `/share` skill for Claude Code that takes a completed WezTerm feature and drafts content for sharing -- blog posts for joshmedeski.com and tweets with relevant hashtags.

- Create a `/share` slash command for Claude Code
- Generate a blog post draft for joshmedeski.com showcasing the feature
- Generate a tweet with the user's usual hashtags
- Pull context from the feature's project note and code changes
