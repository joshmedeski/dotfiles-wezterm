# YouTube Thumbnail Wallpapers

A standalone CLI that fetches thumbnails from recently watched YouTube videos via the YouTube API, with WezTerm integration to use them as wallpapers.

- Build a separate CLI tool that authenticates with the YouTube API
- Fetch recently watched videos and extract thumbnails
- Filter videos by music category for album-art-style wallpapers
- Use Stable Diffusion to remove text overlays from thumbnails
- Search SearXNG or similar to find cleaner versions of the same image
- Allow storing processed images in the wallpapers directory
- Integrate with WezTerm's wallpaper system as a new image source
